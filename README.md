# pregit

My git commits are messy. I want to give them more structure and consistency.

## Install

### Prerequisites

- Go 1.18 or higher.
- Git initialized in repo.

### Build

1. Clone the repo:

   ```
   git clone https://github.com/dnlaxe/pregit.git
   cd pregit
   ```

2. Build + install the binary:

   ```
   make install
   ```

   This compiles the tool and moves it to `/usr/local/bin`, allowing you to run pregit from any directory.

### Use

Navigate to any git-initialized directory and simply run `pregit`
